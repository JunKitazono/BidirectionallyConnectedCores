This folder contains the codes for reproducing the simulations in Kitazono et al., 2020.

HierarchicalStructure.m
Find complexes of a simple exemplary model (Section 9.1).

ComputationTime.m
Measure the computation time of searching for complexes by simulations (Section 9.2).

ECoGDataAnalysis.m
Apply Hierarchical partitioning for complex search (HPC) to ECoG data (Section 9.3).

See each m-file for more details.
